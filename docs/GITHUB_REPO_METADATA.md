## About text (short)

Pico 2W Wi‑Fi lamp with Web UI RGB control and colorful WS2812 presets.

## Description (longer)

Raspberry Pi Pico 2W firmware that creates a local AP (`picoled`) and serves a phone-friendly Web UI at `http://192.168.4.1` for manual color control, brightness, and animated LED presets including Classic Vacuum Tube Amber.
